    About The Project

    This project was developed as my (Elliot Dean) Wireless Sensor Networks (CSEE 4240) Spring 2014 Term Project.

    The purpose of this project is to allow mobile robots to be directed to take temperature and humidity readings within a desert environment without persistent access to the internet or any other form of communication. Readings taken by motes within the network can be searched for by using the Find Data option. Upcoming coordinates and other parameters for individual groups within the network can be set by using the Set Locations option

    The layout of this page is based on an HTML template downloaded from Templatemo.com

    The ability to move markers on the map in the Set Locations section was adapted from source code downloaded from RectangleWorld.com
